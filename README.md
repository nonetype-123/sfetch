# Installation
Write the following commands:
```shell
git clone https://github.com/nonetype-123/simplefetch/
cd simplefetch
chmod +x setup.sh
./setup.sh
```
!IMPORTANT!

Works only with bash. Zsh and other shells do not work. This problem will be fixed in the future!
